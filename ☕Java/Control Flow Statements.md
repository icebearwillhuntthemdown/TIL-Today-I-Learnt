# Control Flow Statements

1. Switch : To test **a certain variable** for multiple values
```java
Switch(varToValidate){
  case condition1 :
    //code to be executed
    break;
  case condition2 :
    //code to be executed
    break;
  case condition3: case condition4: case condition5:  //able to concatenate! 
    //code to be executed
    break;
  default:
    //code to be executed
    break;
}
```
:bulb: 복수의 조건을 연달아 쓸 수 있단 것을 알게 됐다.
